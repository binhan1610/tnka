THIỆP MỜI LỄ TỐT NGHIỆP - ĐẶNG BÌNH AN

Cách chạy:
1. Giải nén toàn bộ thư mục.
2. Giữ nguyên cấu trúc các file và thư mục assets.
3. Mở file index.html bằng Chrome, Edge hoặc Firefox.

Chỉnh thông tin sự kiện:
- Mở script.js.
- Sửa object eventData ở đầu file.

Thay ảnh:
- Thay assets/avatar.png bằng ảnh mới, giữ nguyên tên file.

Thay nhạc:
- Thay assets/music.mp3 bằng file MP3 mới, giữ nguyên tên file.

Lưu ý:
- Website không cần npm, framework hoặc server.
- Google Fonts sẽ dùng font dự phòng nếu máy không có Internet.
- Lời chúc được lưu trong localStorage của trình duyệt, không gửi lên máy chủ.
